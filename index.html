This is a JQuery plugin that allows you to create an high quality justified gallery of images.
A common problem, for people who create sites, is to create an elegant image gallery that manages the various sizes of thumbnails. Flickr and Google+ manage this situation in an excellent way, the purpose of this plugin is to give you the power of those solutions, with a new professional and open source plugin.
